Q1: What are the different ways to declare a variable in JavaScript? var, let, const ‚Äì var is function-scoped, let & const are block-scoped. const cannot be reassigned.
Q2: What is the difference between == and === in JavaScript? == checks only value (type conversion), === checks both value and type.
Q3: What is a JavaScript function and how do you define one? A function is a block of reusable code. Defined using function keyword or arrow function.
Q4: What are arrow functions in JavaScript? Short syntax using =>, no own this, often used for callbacks.
Q5: What is the difference between null and undefined? undefined ‚Üí variable declared but not assigned.
null ‚Üí explicitly assigned ‚Äúno value.‚Äù
Q6: What are JavaScript data types? Primitive: String, Number, Boolean, Null, Undefined, Symbol, BigInt.
Non-Primitive: Object, Array, Function, Date, Map, Set, etc.
Q7: What is the difference between function declaration and function expression? Declaration ‚Üí hoisted, can be called before definition.
Expression ‚Üí assigned to variable, not hoisted.
Q8: What is hoisting in JavaScript? JS moves variable and function declarations to the top of scope before execution.
Q9: What is the difference between synchronous and asynchronous JavaScript? Synchronous ‚Üí executes line by line (blocking).
Asynchronous ‚Üí non-blocking (setTimeout, fetch, Promises).
Q10: What are JavaScript Promises? Objects that represent a value which may be resolved (fulfilled) or rejected in the future.
Q11: What are JavaScript Closures? A closure is when a function remembers variables from its outer scope even after the scope has finished.
Q12: What is the difference between call, apply, and bind? call() ‚Üí calls function with arguments individually.
apply() ‚Üí calls function with arguments as array.
bind() ‚Üí returns new function with bound this.
Q13: What is the difference between localStorage, sessionStorage, and cookies? localStorage ‚Üí permanent storage.
sessionStorage ‚Üí clears when tab closed.
cookies ‚Üí stored with expiry, sent to server.
Q14: What is the Event Loop in JavaScript? The event loop manages execution of synchronous and asynchronous tasks. It pulls tasks from the callback queue and executes them when the call stack is empty.
Q15: What is async/await in JavaScript? async/await is syntactic sugar for Promises. async makes a function return a Promise, await pauses execution until the Promise resolves.
Q16: What is the difference between for...in and for...of? for...in ‚Üí iterates over keys (properties) of an object.
for...of ‚Üí iterates over values of iterable objects like arrays, strings.
Q17: What is the difference between deep copy and shallow copy? Shallow copy ‚Üí copies references (changes affect original).
Deep copy ‚Üí creates completely new copy of object/array.
Q18: What is a JavaScript prototype? Prototype is an object from which other objects inherit properties. Used for inheritance in JavaScript.
Q19: What is DOM in JavaScript? DOM (Document Object Model) is a programming interface for HTML/XML documents. It allows JS to manipulate structure, style, and content.
Q20: What are higher-order functions? Functions that take other functions as arguments or return a function. Example: map, filter, reduce.
